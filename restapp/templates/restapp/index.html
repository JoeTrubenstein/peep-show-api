Peep Show API

moreish data for your next web project

What is this?

This is a small educational project intended to demonstrate the process of developing an API.

It's written in Python, and utilizes both Django and the Django Rest Framework.

It allows you to fetch Peep Show data to use in your development projects, like so:

Jeremy Usbourne

I guess it was just one of those freaky urban things, like those people that go on fire for no reason.

Super Hans
